できること

  • コーディング

    HTML/CSS、JavaScriptを用いたWebコーディングに対応。 FigmaやPhotoshop、Illustratorなどのデザインカンプから、 細かなディテールや空気感までシームレスに再現します。 滑らかな動的演出の実装もお任せください。

    • HTML
    • CSS/SCSS
    • jQuery
    • JavaScript

  • レスポンシブ

    モバイルファーストの現代に欠かせない、各種デバイス（スマホ・タブレット・PC）への表示最適化を行います。 端末を選ばず「見やすく、操作しやすい」デザインを保つことで、ユーザーの閲覧ストレスを解消。 動線や視認性に配慮した丁寧なレスポンシブを実装いたします。

  • Wordpress

    WordPressを用いた動的なサイト構築に対応。デザインの自由度が高いオリジナルテーマの制作から、 既存の静的サイトを活かしたWordPress移行まで幅広く承ります。 お知らせや主要コンテンツのパーツ化など、日々の更新作業がストレスなく行える設計を徹底します。

  • webデザイン

    Figmaを使用し、サイトの目的やブランドの空気感に合わせた最適なビジュアルをご提案します。 全体のトーン＆マナーの設計から、レイアウト、細かなパーツのデザインまで丁寧に作り込みます。 デザインからコーディングまで一貫したクオリティを保ち、魅力を最大限に引き出すWebサイトの土台を構築します。
